# Source vs. raw vs. derived data

Source data: data before harmonization, reconstruction, and/or file format conversion (for example, E-Prime event logs or DICOM files)

Raw data: unprocessed or minimally processed due to file format conversion data

Derived data: processed data

for more details please refer to [BIDS](https://bids-specification.readthedocs.io/en/stable/common-principles.html#source-vs-raw-vs-derived-data).
